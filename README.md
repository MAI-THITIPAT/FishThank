# FishThank
A project for CPE241 | INTERNET OF THINGS

This project allows users to monitor environmental data and control home appliances remotely through the Blynk mobile application.

    Microcontroller: ESP8266 (NodeMCU/WeMos D1 Mini)
    IoT Platform: Blynk IoT (Mobile App & Cloud)
    Language: C++ (Arduino IDE)
    Communication: Wi-Fi (IEEE 802.11 b/g/n)
    Features: - Real-time sensor data visualization on Blynk Dashboard.
        Remote switch control for appliances (Relay modules).
        Mobile notifications and alerts.
